Built a 3D product configurator for Hulck, a Dutch manufacturer of heavy-duty industrial trailers. B2B buyers configure their trailer — selecting chassis type, body length, axle configuration, accessories, and 20+ further attributes — with real-time 3D updates and a quote request submitted directly through WooCommerce.

Technical Implementation:
- Integrated Expivi's 3D rendering and CPQ (Configure, Price, Quote) platform with WordPress and WooCommerce
- Supported 23+ configurable attributes covering chassis, body, axle, and accessory combinations
- Implemented high-resolution screenshot capture (1920×1080) of the configured trailer for quote documents
- Connected configuration output to WooCommerce for structured quote requests and order intake
- Built for B2B buyers who need precise spec control before committing to a purchase

Key Features:
- 360° real-time 3D trailer visualization
- 23+ configurable attributes with live model updates
- High-resolution screenshot export for quotes
- WooCommerce-backed quote request flow

The project is completed and online. Configure a trailer at [hulck.com](https://hulck.com/stel-zelf-uw-industriele-aanhanger-samen/).
