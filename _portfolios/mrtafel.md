Built three interactive 3D product configurators for MrTafel, a Dutch maker of custom solid-oak furniture. Customers configure their dining table, coffee table, or chair — choosing dimensions, wood type, leg style, and finish — with live pricing updating in real time and an AR preview to see the piece in their own space before ordering.

Technical Implementation:
- Integrated Expivi's 3D rendering platform with WordPress and WooCommerce for seamless add-to-cart and checkout
- Implemented live pricing engine responding to every configuration change (dimensions, material, finish)
- Enabled AR preview so customers can place configured furniture in their own room via mobile
- Built three separate configurators sharing the same integration pattern: dining tables, coffee tables, and chairs
- Connected configuration state to WooCommerce product options for accurate order processing

Key Features:
- Real-time 3D visualization with material and finish swaps
- Dynamic pricing updated on every option change
- Augmented reality room preview on mobile
- Direct WooCommerce integration — configured item added to cart with full spec

The project is live. Try the configurators at [mistertafel.nl](https://mistertafel.nl/zelf-samenstellen-eettafels/).
