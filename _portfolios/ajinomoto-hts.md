Developed Ajinomoto HTS (Handy Terminal System), a native Android application for Ajinomoto Sales Indonesia's field sales teams. The app supports daily sales operations across the country, processing hundreds of millions of rupiah in transactions per day. Released as an internal tool.

Technical Implementation:
- Built with Kotlin for Android Native targeting sales teams operating across Indonesia
- Implemented offline-first architecture to ensure reliability in areas with poor connectivity
- Integrated with Ajinomoto's internal backend systems for real-time order and transaction data
- Handled complex daily transaction workflows for field sales representatives

Key Technical Achievements:
- Maintained stable offline data sync for multi-region sales operations
- Supported high-volume daily transaction processing with data integrity guarantees
- Delivered robust and reliable UX for non-technical field sales users

> Released Internally — not available on Play Store or App Store.
